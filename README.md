## Description
This is a plugin for the B2BKing premium plugin to enable a more advanced dashboard for the B2B users. It extends the core capabilities to provide a cleaner, more feature-rich area for wholesale customers to manage their account, orders, and more.

## Features
- **Shop Filters**: Adds filters to the shop page to filter products by best sellers, new products, and discounts.
- **Bulk Actions**: Adds bulk actions to the shop page to bulk delete products.
- **Variations Columns**: Adds variations columns to the shop page to show variations of products.
- **Guest Info Table**: Adds an information table without prices for guest users.
- **B2B Info Table**: Adds an information table with prices for B2B users.
- **User Report**: Adds a report section in the user account to show sales charts and most popular products.

## Installation
1. Ensure **B2BKing Premium** is installed and active.
2. Upload this plugin to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
